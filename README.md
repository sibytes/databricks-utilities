# Development Setup

Create virual environment and install dependencies for local development:

```
python3 -m venv venv
source venv/bin/activate
pip install --upgrade pip
pip install -r requirements.txt
```


# Build

Build python wheel:
```
python setup.py sdist bdist_wheel
```
